# ⚛️ React Practice Projects

A collection of small projects and exercises to practice and strengthen React skills. Each folder contains an independent project focused on specific concepts or features of the library.

---

## 🎯 Goal

The purpose of this repository is to learn and experiment with React through hands-on projects, covering different levels of complexity and a variety of use cases.

---

## 🗂️ Repository Structure

```
react-practice/
├── 01-shopping-card/
├── 02-tic-tac-tpe/
├── 03-gifs-app/
├── 04-hook-app/
└── ...
```

Each project has its own `README.md` with a brief description and instructions on how to run it.

---

## 🧠 Topics Covered

- Components and Props
- State with `useState`
- Side effects with `useEffect`
- Context API
- React Router
- Custom Hooks
- Forms and validation
- Consuming external APIs
- Global state management (Redux / Zustand)

---

## 🚀 Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/react-practice.git
   ```

2. Navigate to the desired project:

   ```bash
   cd react-practice/01-counter-app
   ```

3. Install dependencies and run the project:
   ```bash
   npm install
   npm run dev
   ```
